# Production Scheduler Hasura

## Overview
Production Scheduler Hasura is a React TypeScript application designed to manage and schedule production tasks efficiently. This project leverages Hasura for real-time data management and GraphQL capabilities.

## Features
- Real-time data synchronization with Hasura
- TypeScript for type safety and better development experience
- Modular architecture with components, hooks, and utilities
- Global styles for consistent design

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/production-scheduler-hasura.git
   ```
2. Navigate to the project directory:
   ```
   cd production-scheduler-hasura
   ```
3. Install dependencies:
   ```
   npm install
   ```
   or
   ```
   yarn install
   ```

### Running the Application
To start the development server, run:
```
npm run dev
```
or
```
yarn dev
```
Open your browser and navigate to `http://localhost:3000` to view the application.

### Building for Production
To create a production build, run:
```
npm run build
```
or
```
yarn build
```

### Testing
To run tests, use:
```
npm test
```
or
```
yarn test
```
